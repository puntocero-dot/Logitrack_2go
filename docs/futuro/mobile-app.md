# Mobile App (Driver) - Futuro

## Objetivo
App nativa en Flutter para conductores de motos.

## Funcionalidades planeadas
- Login conductor (rol: driver)
- Recibir pedidos asignados
- Actualizar ubicación en tiempo real (geolocation-service)
- Marcar pedido como recogido/en ruta/entregado
- Ver mapa con ruta optimizada
- Historial de entregas del día
